Season 1 info

  • The seed is "3969420520284585470"
  • The world will be split up into different districts, please be aware that failure to follow the districts build style may result in your build being removed (within reason).
  • The world border will be at 10k, you do not have to build in the areas with the districts but it is recommended to. You can also build elsewhere on the server that doesn't have a district in any theme you like.
  • If you have a new suggestion for a district, let us know on Discord via opening a ticket!

Graves

The server has a graves plugin, so when you die your stuff will be placed safely inside a grave. It will also save all of your levels. You can also teleport to your grave (if you don't like this then don't use it...) by using the command "/graves". After 30 minutes your grave can be opened by anyone, and after an hour the grave will despawn. Graves will also spawn a zombie once you loot them.

Datapacks and plugins

The server uses the following datapacks:

  • Vanilla Tweaks
  • Vanilla Tweaks [crafting tweaks]
  • Blaze and Caves advancements (custom advancements)
  • Better underwater ruins
  • Better ruined Nether portals

As well as datapacks, the server uses some server side plugins. The majority of these do not effect player experience and are for admin work. You can find a full list of plugins here, however this list is updated each season and may not be correct for season 2.
