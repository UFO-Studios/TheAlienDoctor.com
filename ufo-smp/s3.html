Season 3 info

Early Game Info

  • The Nether will be opened after 1 week of the server opening
  • The End will be open after 1 month of the server opening

World Borders

Players are unable to pass the world border. Please note, Bedrock players will be able to pass due to limiations on Bedrock edition, however they will take damage and not be able to modify any blocks outside the world border.

  • Overworld: 4000 in each direction from 0, 0
  • Nether: 2000 in each direction from 0, 0
  • End: No borders, but regular resets when lots of the outer end islands have been expored

Datapacks

The server uses the following datapacks:

  • Vanilla Tweaks
  • Blaze and Caves advancements

As well as datapacks, the server uses some server side plugins. The majority of these do not effect player experience and are for admin work. You can find a full list of plugins here, however this list is updated each season and may not be correct for season 3.

Other Info

  • Graves: Everytime you die, your items will be put into a grave that can survive in the void and lava. Graves expire after 5 minutes and cannot be looted by other players. You can view the location of your current graves via the command /graves.
  • PvP Toggle: To stop unwanted PvP, it is off by default. To enable it, type /pvp on. To disable, type /pvp off. Your name will turn red if you have PvP on (although this may not show on Bedrock's player list).
