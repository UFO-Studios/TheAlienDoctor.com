Minecraft Resource Packs

What are Resource Packs?

Minecraft resource pack can change the way the game looks, through modifying .png files and .json files. Unlike behaviour packs resoure packs are client side only, which means only the clients with the resource pack applied can see it. They can also optionally be applied to a world/server, which means everyone on that world/server will be able to see the resource pack. As well as this, they will not disable achivements.

How to install Resource Packs

Alien's Resource Packs

Most of Alien's resource packs are for Minecraft Bedrock edition, as that is the version that he plays. If you wish to try one on Java edition, then try out Alien's pack converter.

Minecraft version: Bedrock 1.18.0+
