BDS Companion - A UI for BDS

Download

Support Alien on Patreon! View on GitHub

A GUI and companion software for the Minecraft Bedrock Dedicated Server Software.

Setup:

Download the exe from the releases page, and move it to an empty folder. Extract a copy of MCBE's BDS to a folder named BDS within the folder the exe is in. Then double click the exe to open! You can find the latest release of BDS here. Please note that this application only supports Windows.
