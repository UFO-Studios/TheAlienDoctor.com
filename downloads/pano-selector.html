Download latest version (MCBE 1.20.0+) [V1.4.5]

Support Alien on Patreon! View on GitHub

Description

This resource pack allows you to select a new menu pano screen from a selection of panos from older Minecraft versions, and hopefully in the future will allow you to add custom panos.

Version 1.2.2
  • Removed the following panos:
    • Ocean-side extreme hills
    • Lakeside sunset
    • Underground lava pool
    This is because these panos were originally taken from the Java edition resource pack Vanilla Tweaks. Since then, the unofficial Bedrock port, Bedrock Tweaks have released them to download on their version of vanilla tweaks. Because of this and so my pack does not get too cluttered, I have decided to remove them from my pack but if you still want them then you can download them on the Bedrock Tweaks website.
  • Added new pack logo
  • Updated the pack version numbering to match my other packs
  • The minimum supported version is now 1.16
