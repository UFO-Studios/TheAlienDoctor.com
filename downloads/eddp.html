Description

This is a simple python app that takes data generated by Elite:Dangerous, and sends it to discord. To use this,either double click the EXE to run, or copy it to "C:\Users\YOUR_USERNAME\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup" . Replace "YOUR_USERNAME" with your window username. (This can be found by checking the "C:/Users" folder for you account) You can find & report issues here

Windows might flag this as unsafe, this is because it has not been signed by Microsoft. You can check out the source code here, and you are welcome to add any code!
