Piglin to Pigman

Showcase video

Download latest version (MCBE 1.16+) [V1.0.2] Download Java port (Java 1.18+)

Description

Piglin to Pigman is a simple pack to change the 1.16 Zombie Piglins back to there pre 1.16 Zombie Pigman form. This inlcudes the model (if using the Java port, optifine is needed for custom models), as well as the texture.
