Description

Alien's wallpaper pack is a collection of screenshots of Alien's series/worlds and builds that can be used as wallpapers! Almost every screenshot is in 1440p so that it looks good on almost any display!

How to use

Simply unzip the .zip file and either set your computers wallpaper to slideshow through all the photos in the folder, or set a screenshot you like to be your static background :D
