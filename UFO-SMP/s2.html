Season 2 info

  • This season features custom Overworld, Nether and End generation via datapacks, as well as some custom structures. At spawn there is a custom village, so it could be a cool idea if people claim/build a house there and we build a spawn village.
  • The seed is "Sub2Alien" or "626336725".
  • As per usual, PvP is toggleable using "/pvp ["on" or "off"]".
  • The server also has graves. You can teleport to graves using "/graves".

World Borders

Players are unable to pass the world border. Please note, Bedrock players will be able to pass due to limiations on Bedrock edition, however they will take damage and not be able to modify any blocks outside the world border.

  • Overworld: 4096, -4096
  • Nether: 2048, -2048
  • End: 4096, -4096

Pruning

Pruning is when chunks are reset. Next time you visit a pruned chunk, it will be reset and regenerate back to how it looks before anyone has changed it. This helps the worlds file size.

All chunks that have been loaded for less than 2 minutes will be pruned in The Overwold and The Nether.

All chunks past 500 500 in The End will also be reset, meaning people will be able to find unraided End Cities more easily.

Datapacks and plugins

The server uses the following datapacks:

  • Terralith (custom overworld gen)
  • Incendium (custom world gen)
  • Nullscape (custom End gen)
  • Blaze and Caves advancements (custom advancements)
  • Jungle and Swamp villagers
  • Better underwater ruins
  • Better ruined Nether portals

As well as datapacks, the server uses some server side plugins. The majority of these do not effect player experience and are for admin work. You can find a full list of plugins here, however this list is updated each season and may not be correct for season 2.
